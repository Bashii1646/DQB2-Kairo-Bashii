# マグマエリアの応用

マグマエリアの仕様を利用して、マグマの流れを自由に制御する方法を検証しました。

## Xでの紹介

<Tweet id="2077968840068358355" />

## 動作

1. 蒸発しない場所に水を配置する
2. 流れているマグマへ水を流す
3. 接触した部分が赤山岩へ変化する
4. 水を止めると同時に、その場所をマグマエリア化する
5. 赤山岩が時間経過でマグマ源へ変化する
6. 新しく生成されたマグマ源から、さらに先へマグマが流れる

## マグマを止める

マグマエリアを無効化したあとに再び水を流すと、

- マグマ源は赤山岩へ戻る
- マグマの流れはそこで止まる

ことを確認しました。

## 特徴

マグマそのものを増減させるのではなく、マグマエリアを切り替えることでマグマ源の生成を制御できます。

そのため、

- 必要なときだけマグマを延長する
- 任意の場所でマグマを止める

といった制御が可能になります。

## 応用

マグマエリアのON/OFFと組み合わせることで、マグマの流れを自由に切り替えるギミックへ発展させることができます。

マグマの経路を動的に変更する仕組みなど、さまざまな応用が期待できます。
