# マグマとマグマエリアの仕様

マグマとマグマエリアの関係について検証しました。

マグマエリア内では、マグマや赤山岩が時間経過によって変化し、マグマ源が生成されます。

## マグマの変化

マグマの中にブロックを置くと、そのブロックはマグマ岩になります。

また、水が接触すると赤山岩などへ変化します。

さらに、そのいずれかがマグマエリア内（またはその付近）にあると、時間経過によってマグマ源へ変化します。

## マグマ源にならない条件

マグマエリアから離れた場所では、そのままの状態を維持し、マグマ源には変化しません。

## 特徴

ここでいう「マグマ化」とは、見た目が変わることではなく、**マグマ源になること**を指します。

マグマ源になると、その場所を起点としてマグマが広がっていくため、ギミックへの影響が大きくなります。

## 応用

マグマエリアをON/OFFすることで、

- 水の蒸発
- マグマ源の生成

を必要なタイミングだけ切り替えることができます。

この仕組みを利用することで、さまざまなマグマギミックへ応用できます。
