# マグマとマグマエリアの仕様

マグマとマグマエリアの関係について検証しました。

マグマエリア内では、マグマや赤山岩が時間経過によって変化し、マグマ源が生成されます。

<Tweet id="2077967431730123004" />

## マグマの変化

マグマの中にブロックを置くと、そのブロックはマグマ岩になります。

また、水が接触すると赤山岩などへ変化します。

さらに、そのいずれかがマグマエリア内（またはその付近）にあると、時間経過によってマグマ源へ変化します。

## 液体を通すアイテムとの関係

続いて、マグマがブロック化する条件を調べました。

<Tweet id="2078138322380316978" />

通常、マグマは水などの液体に触れると赤山岩ブロックへ変化します。

しかし、**あらかじめその場所に液体を通すアイテムが設置されている場合は、ブロック化しません。**

確認できたアイテムの例

- 木のさく
- マグネブロック
- その他、液体を通すアイテム

この性質を利用することで、マグマの流れを維持したまま水と接触させることができ、今までとは異なるギミックや回路への応用が期待できます。

## マグマ源にならない条件

マグマエリアから離れた場所では、そのままの状態を維持し、マグマ源には変化しません。

## 特徴

ここでいう「マグマ化」とは、見た目が変わることではなく、**マグマ源になること**を指します。

マグマ源になると、その場所を起点としてマグマが広がっていくため、ギミックへの影響が大きくなります。

## 応用

マグマエリアをON/OFFすることで、

- 水の蒸発
- マグマ源の生成

を必要なタイミングだけ切り替えることができます。

また、液体を通すアイテムを組み合わせることで、マグマと水を共存させた新しいギミックや回路への応用も期待できます。
