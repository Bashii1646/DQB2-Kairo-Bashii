# マグマエリアを利用した選択回路

マグマエリアの「液体を蒸発させる」性質を利用し、複数の回路の中から必要な場所だけを選択して動作させる回路を作成しました。

<Tweet id="2080337845454254196" />

## 動作

この回路では、2段階の選択方式を採用しています。

1. マグマ選択スイッチで動作させるエリアを選択します。
2. 水車選択スイッチで、そのエリア内の水車を選択します。

これにより、複数の回路へ同時に信号を送っても、選択した場所だけが動作します。

## 特徴

マグマエリアによって不要な水を蒸発させることで、必要な経路だけに水を流すことができます。

また、水車は最後の出力で1回だけ使用するため、選択してから約10秒で結果を取り出せます。

## 応用

この仕組みを利用することで、

- 回路の選択
- メモリの選択
- 計算機の入力切り替え
- 複数回路の切り替え

などへ応用できます。
