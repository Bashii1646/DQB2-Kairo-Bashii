# マグマエリアのON/OFFスイッチ

マグマエリアの判定を利用し、スイッチ1つでマグマエリアを瞬時にON/OFFできる装置を作成しました。

## Xでの紹介

## 仕組み

前回検証したマグマエリアの判定を利用し、エリア判定を切り替えることで、マグマエリアを瞬時にON/OFFしています。

マグマそのものを増減させる必要がないため、マグマ源を増やしてしまうことなく、安定して切り替えることができます。

## 動作

マグマエリアをONにすると、

- 水が蒸発する

マグマエリアをOFFにすると、

- 水が蒸発しなくなる

この2つの状態をスイッチ1つで切り替えられます。

## 特徴

- マグマ源を増減させない
- スイッチ1つで瞬時に切り替え可能
- 水の蒸発を自由に制御できる

マグマエリアの仕組みを利用することで、さまざまなギミックへの応用が期待できます。

## 注意点

この装置を動作させるエリア化ギミックは、対象エリアの最上段に設置する必要があります。

また、上下方向に離れた場所にある液体は、プレイヤーが近づくまで蒸発が反映されない場合があることも確認しています。
