# マグマエリアと水車のギミック

マグマエリアの仕様と水車を組み合わせたギミックを作成しました。

<Tweet id="2077977882421575769" />

## 動作

1. マグマへ水を流し、赤山岩へ変化させる。
2. その場所をマグマエリア化すると、赤山岩が時間経過でマグマ源になる。
3. 新しく生成されたマグマ源から、もともと届かなかった場所までマグマが流れる。
4. マグマエリアを解除し、届いたマグマへ別の水を流す。
5. マグマが赤山岩となって通路ができ、水車まで水が到達する。

## 特徴

マグマそのものを直接制御するのではなく、マグマエリアのON/OFFによってマグマ源の生成を制御しています。

これにより、必要なときだけマグマの流れを延長し、その後は水路へ戻して水車を動作させることができます。

## 応用

マグマと水の変化を組み合わせることで、

- 水路の開閉
- 水車の起動
- 条件付き通路の作成

など、さまざまなギミックへ発展させることができます。
