# マグネブロックを利用した回路の土台

マグマの「液体を通すアイテムがある場所ではブロック化しない」という性質を利用し、マグネブロックを回路の土台として利用する方法を検証しました。

<Tweet id="2078140539741073737" />

## 動作

1. マグネブロックの位置では、マグマと水が重なってもブロック化しません。
2. 液体がないタイミングでマグネブロックを移動します。
3. マグネブロックがなくなった位置がブロック化します。
4. 生成されたブロックの上を水が流れ、新たな経路へ進みます。

## 特徴

これまでマグネブロックは、ブロックの移動や出力ギミックとして利用することが多くありました。

この方法では、マグネブロックを**回路の土台**として利用できることを確認しました。

## 注意点

ブロック化すると元の状態へ戻すことはできません。

そのため、同じ場所で繰り返し使用することはできず、基本的に一度だけ動作するギミックとなります。

## 応用

- 一度だけ動作する回路
- マグネブロックを利用した状態保持
- マグマのブロック化を利用したギミック
- 新しい経路を作る回路
