# 水車ワンショット回路

水車を使って、最初の1回だけ信号を出力する回路です。

## Xでの紹介

<Tweet id="2076273865056284738" />

## 動作

1. 水門を開くと水が流れる
2. 水流で水車が回る
3. 水車がスイッチを押し、出力先へ信号を送る
4. 同時にもう一方のスイッチを押して、水門を閉じる
5. 水流が止まり、水車も停止する

## 特徴

水車は水が流れ続けていると、およそ10秒ごとに繰り返しスイッチを押します。

この回路では、水車が最初にスイッチを押したとき、その動作を利用して水門を閉じることで、出力を最初の1回だけに絞っています。
