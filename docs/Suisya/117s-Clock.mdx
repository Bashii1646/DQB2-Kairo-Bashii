# 11.7秒周期の水車クロック回路

水門を利用して、水車の動作周期を約11.7秒まで延長したクロック回路です。

<Tweet id="2078056338513707486" />

## 動作

1. 水門を開いて水を流す
2. 水流によって水車が動作する
3. 水車から信号を出力する
4. 水門の開閉によって水流を制御する
5. 水車が再び動作する

この動作を繰り返すことで、約11.7秒ごとに信号を出力します。

## 特徴

通常の水車は、水が流れ続けているとおよそ10秒ごとにスイッチを押します。

この回路では、水門を利用することで、水車の周期を約11.7秒まで延長しています。

13.25秒周期のクロック回路とは異なる周期になるため、組み合わせることでさらに長い周期を作ったり、動作タイミングを調整したりできます。
