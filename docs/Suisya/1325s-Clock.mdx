# 13.25秒周期の水車クロック回路

扉が自動で閉じる性質を利用して、水車の動作周期を約13.25秒まで延長したクロック回路です。


<Tweet id="2077001714708963518" />

## 動作

1. 水を流して水車を動作させる
2. 水車から信号を出力する
3. 扉が開くことで、一時的に水車の動作を止める
4. 一定時間後に扉が自動で閉じる
5. 水車が再び動作する

この動作を繰り返すことで、約13.25秒ごとに信号を出力します。

## 特徴

通常の水車は、水が流れ続けているとおよそ10秒ごとにスイッチを押します。

この回路では、扉が一定時間後に自動で閉じる性質を利用することで、水車の周期を約13.25秒まで延長しています。

異なる周期のクロック回路と組み合わせることで、さらに長い周期を作ったり、動作タイミングをずらしたりできます。
