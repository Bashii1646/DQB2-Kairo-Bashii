# 2進数の繰り上げ機構

以前紹介した「指定回数入力回路」を利用し、2回入力されたら次の桁へ信号を送るようにすることで、2進数の繰り上げ機構を作成しました。

<Tweet id="2078845894724919770" />

## 動作

各桁は2回入力されると、次の桁へ信号を送ります。

この仕組みを並列化することで、複数桁の2進数に対応した繰り上げ処理を実現できます。

また、1回だけ入力された場合は別の水路へ流れるようにし、最後に出力回路で各桁を読み取ることで、2進数の計算結果を取り出せます。

## 特徴

以前紹介した指定回数入力回路を応用することで、2進数の繰り上げ処理を実現できます。

水車を利用しているため動作には時間がかかりますが、保持回路や出力回路と組み合わせることで、計算回路へ発展させることができます。

## 動作時間

繰り上がりが発生するたびに、水車の動作時間が必要になります。

例えば、

```
1111
+0001
-----
10000
```

のように連続して繰り上がる場合は、1桁あたり約10秒＋αかかるため、およそ50秒ほどで計算が完了します。

## 応用

- 2進数の繰り上げ処理
- 多桁演算
- 2進数加算器
- 計算回路
