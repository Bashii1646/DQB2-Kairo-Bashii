# 保持出力継続・ワンショット出力回路

保持回路を並列化することで、保持した状態を継続して出力したり、一度だけ出力したりできる回路です。

<Tweet id="2078513432375296470" />

## 動作

保持回路の毒水用のドアの上に水門を設置し、さらに並列化することで、保持した場所を出力として利用できます。

### 継続出力

- 毒扉：出力を一時停止
- 水扉：出力を再開
- 毒水門：その場所だけ出力を停止

保持した状態を必要な間だけ出力し続けることができます。

### ワンショット出力

水車の出力を毒水門へフィードバックすることで、出力を一度だけに制限できます。

## 特徴

同じ基本構成で、

- 継続出力
- ワンショット出力

を切り替えられるため、用途に応じた使い分けが可能です。

## 応用

- 保持状態の読み出し
- 継続出力回路
- ワンショット出力回路
- 論理回路
- 計算回路
