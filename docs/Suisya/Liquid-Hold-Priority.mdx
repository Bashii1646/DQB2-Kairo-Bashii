# 液体の保持判定

補助液体を流したときに、保持対象の液体が保持されるかどうかを、全7種類の液体の組み合わせで検証しました。

<Tweet id="2079776238751936759" />

## 検証方法

保持対象となる液体に対して、別の液体を補助液体として流し込み、保持されるかどうかを確認しました。

全7種類の液体について、すべての組み合わせを検証し、結果を一覧表にまとめています。

## 検証結果

単純な優先順位で保持されるわけではなく、液体の組み合わせによって挙動が異なることを確認しました。

特に、泥水だけが例外となる組み合わせがあり、単純な優先順位では説明できない仕様となっています。

詳しい結果は掲載している一覧表をご覧ください。

## 応用

液体ごとの保持判定を理解することで、

- 保持回路
- 液体を利用した論理回路
- 水車ギミック

などを、より安定して構築できるようになります。
