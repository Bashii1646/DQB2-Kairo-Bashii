# ワンショット保持・リセット回路

ドアを利用することで、ワンショット入力だけで保持とリセットを行える回路です。

<Tweet id="2078460512141259123" />

## 動作

### 保持

1. ドアを開き、一瞬だけ水を放水します。
2. 放水後は水の状態が保持されます。

### リセット

1. リセット側のドアを開きます。
2. 毒水側に穴が作られ、保持状態が解除されます。

## 特徴

水門の代わりにドアを使用することで、ワンショット入力だけで保持とリセットを実現できます。

そのため、複雑なタイミング制御を行わなくても、状態の保持と解除を簡単に行えます。

## 注意点

水通常まきと毒水浅瀬まきは、配置する距離を適切に調整する必要があります。

近づけすぎると正常に動作しない場合があります。

## 応用

- リセット回路
- 状態記憶
- 論理回路
