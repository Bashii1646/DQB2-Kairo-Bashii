# シンプルな保持回路

毒水浅瀬まきと水通常まきを組み合わせることで、水の状態を保持できる回路を作成しました。

<Tweet id="2078446110268621299" />

## 動作

1. 毒水側の水門を開いた状態で水を流します。
2. 水が流れたあと、水側の水門を閉じます。
3. 水門を閉じても、水はそのまま保持され続けます。

## 特徴

毒水浅瀬まきと水通常まきの性質を組み合わせることで、シンプルな構成で状態保持を実現できます。

複雑な回路を必要としないため、さまざまな回路の基礎として利用できます。

## 注意点

毒水浅瀬と水は、近づけすぎると毒水浅瀬が消えてしまいます。

そのため、両者の間には少し距離を空けて配置する必要があります。

## 応用

- 状態保持回路
- 記憶回路
- 論理回路の基礎
- 複雑なギミックへの応用
