# 段階選択回路

手前から順番にスイッチを押さないと先へ進めない回路です。

各スイッチが現在の段階を表すため、段階選択や認証などに利用できます。

<Tweet id="2076973600138244158" />

## 動作

1. 手前のスイッチから順番に押す
2. 押した位置まで進めるようになる
3. 奥まで進んだあとでも、手前のスイッチを押すとその段階まで戻る
4. 各スイッチが現在の段階を選択するように動作する

## 特徴

通常の順番認証とは異なり、一度進んだあとでも前の段階へ戻すことができます。

そのため、単なる認証だけでなく、現在の状態を切り替える回路としても利用できます。

## 応用

- 段階選択
- 順番認証
- パズルギミック
- 状態切り替え回路
