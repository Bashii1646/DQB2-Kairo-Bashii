# 水流段階停止回路

普段は一番奥まで水が流れ続けている状態から、各スイッチによって停止する範囲を切り替える回路です。

<Tweet id="2076979341226057788" />

## 動作

1. 通常時は、一番奥まで水が流れ続ける
2. 各スイッチを押すと、その位置から奥の水流が一時的に止まる
3. 水流が止まった範囲の水車も停止する
4. 奥のスイッチでは奥側だけが停止する
5. 手前のスイッチでは、その位置から奥すべてが停止する

## 特徴

押したスイッチの位置によって、水流と水車が停止する範囲が変わります。

そのため、複数の動作箇所を段階的に停止させたり、動作範囲を切り替えたりできます。

## 応用

- 水車の動作範囲の切り替え
- 段階的な停止制御
- 順番パズル
- 複数区画の一括停止
