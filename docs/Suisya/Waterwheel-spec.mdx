# 水車の仕様

水車の基本的な仕様について検証しました。

水車が回転する液体の種類や、液体を検知する位置など、水車回路を作るうえで重要な仕様をまとめています。

<Tweet id="2078433178587652101" />

## 水車が回る液体

水車が回転する液体は次のとおりです。

### 回る液体

- 水
- 海水
- 泥水
- お湯

### 回らない液体

- 血
- 毒水
- マグマ

## 液体を検知する位置

水車は設置範囲8マスすべてを判定しているわけではありません。

実際に液体を検知するのは**1マスのみ**です。

また、この判定位置は**水車の向きではなく、方角に対して固定**されています。

そのため、水車の向きを変えても、検知する位置は変化しません。

## 応用

この仕様を理解することで、

- 水車回路
- クロック回路
- 保持回路
- 液体を利用した論理回路

などを、意図した位置で安定して動作させることができます。
